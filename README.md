Windows Activation Management Tool
项目描述
这是一个基于Python语言写的的Windows激活管理工具，提供图形化界面来管理Windows系统的激活状态。该工具集成了多种激活相关功能，包括状态检测、密钥管理、OEM信息查询等。

主要功能
🔍 激活状态检测
检查当前Windows激活状态

显示许可证详细信息

查看激活过期时间

获取系统基本信息

🔑 密钥管理
显示已安装的产品密钥

安装新的产品密钥

验证产品密钥格式

安全隐藏完整密钥信息

🏷️ OEM信息查询
查询OEM产品密钥

获取系统制造商信息

读取BIOS信息

检查注册表中的OEM数据

⚡ 激活操作
直接激活Windows

KMS激活（支持自定义服务器）

自动激活尝试

🔄 重置功能
完全重置激活状态

仅卸载产品密钥

仅清除KMS服务器设置

仅重置授权状态
